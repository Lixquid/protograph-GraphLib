# GraphLib

A standard library for ProtoGraph.

## Installation

In your `protograph.toml` file, add:

```toml
[dependencies]
GraphLib = { uri = "https+pgzip://github.com/Lixquid/protograph-GraphLib/releases/download/vMAJOR.MINOR.PATCH/GraphLib.zip" }
```

and use with:

```protograph
from GraphLib use String/PadStart
```
